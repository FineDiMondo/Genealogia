# man status / config / theme (V0)

## status

- `status`

Mostra stato DB, numero tabelle, record aperto nel contesto e ambiente attivo.

## config

- `config show`
- `config set <key> <value>`

Gestisce configurazione runtime lato shell.

## theme

- `theme <nome>`

Applica tema terminale (esempi: `terminal`, `normanno`, `svevo`, `aragonese`, `castigliano`, `risorgimentale`).
